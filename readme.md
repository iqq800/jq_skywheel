# A coverflow-like jquery plugin.

## How to use

html:

```
<ul id="demo">
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
</ul>
```

js:

```
$('#demo').skywheel();
```

## Notice

Requirements: jquery

Browsers: Firefox / Chrome / Safari

There must be >= 5 <li> elements in the list!
